# User-Authentication-System
Production-ready Spring Boot backend with JWT authentication, rate limiting, logging, and AWS EC2 deployment using Nginx and HTTPS.

## 🚀 Features
- JWT Authentication
- Rate Limiting (Login protection)
- Global Exception Handling
- Logging
- Secure Password Hashing (BCrypt)

## ☁️ Deployment
- AWS EC2
- Nginx Reverse Proxy
- HTTPS enabled
- Elastic IP

## 🔐 Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Nginx

## 📌 Endpoints
- POST /auth/register
- POST /auth/login
